Key Features

  • Dashboard Overview: View total orders, pending orders, completed orders, and total revenue at a glance
  • Advanced Search: Find orders by Order ID, customer name, email, or phone number
  • Filter by Status: Filter orders as Pending, Completed, or Cancelled
  • View Details: Click "View" to see complete order information and items
  • Update Status: Change order status from the order details modal
  • Delete Orders: Remove orders from the system permanently
  • Export to CSV: Download all orders as an Excel-compatible CSV file
  • Secure Logout: Log out safely from the admin panel

Order Status Types

Each order has one of the following statuses:

PENDING

New order awaiting processing and confirmation

COMPLETED

Order has been fulfilled and delivered

CANCELLED

Order has been cancelled by customer or admin

Order Details Structure

Each order contains the following information:

Field Description
Order ID Unique identifier starting with #
Customer Name Full name of the person placing the order
Email Customer's email address for contact
Phone Customer's phone number
Items List of products ordered with quantity
Amount Total order value in Rupees (₹)
Status Current status (Pending, Completed, Cancelled)
Date Order placement date and time

Common Actions

Search for an Order

1. Type order ID, customer name, email, or phone in search box

2. Click "Search" button or press Enter

View Order Details

1. Find the order in the table

2. Click the blue "View" button

3. A popup will show all order information

Update Order Status

1. Open order details (click "View")

2. Select new status from dropdown at bottom

3. Status updates automatically

Export Orders

1. Click "Export CSV" button in the toolbar

2. File downloads to your computer (can be opened in Excel)

Logout

1. Click "Logout" button in top-right corner

2. You'll be taken back to login page

Tips & Tricks

Pro Tip: Use the search feature to quickly find an order by customer name instead of scrolling through all orders.
Backup Regularly: Export orders to CSV periodically to create backups of your order data.
Status Management: Keep pending orders at the top by sorting, and update status to "Completed" once delivered.
Browser Cache: Clear browser cache monthly to ensure optimal performance and latest data.
Session Timeout: Your session expires after 24 hours for security. Log in again if needed.
